如何使用这个 img

这个img是直接可以启动的

但是由于是 cloud img，所以一开始无法使用账号密码登陆，也没法通过ssh登陆。
但是可以通过 mount 整个 img，将主机的 ssh的 id_rsa.pub 添加到 cloud img的 authorized_keys中，然后就可以通过 ssh 登陆了。
