perf支持的选项

-e 事件选择器
-p 只跟踪制定线程
-P 采样周期，就是一共采样多久？
-F 频率，就是1s之内记录多少次
-g call_graph，打开调用栈追踪，火焰图就是利用的此特性
-a 从所有cpu搜集数据
-D 频率，就是1s之内记录多少次
-G 只采用 cgroup name内的信息
-C 指定CPU的数据
